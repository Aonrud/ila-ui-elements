Image Viewer

The image viewer on this page is instantiated with the button text symbols replaced with icons, and the close button title altered.

const iv = new ila.ImageViewer({
	showDownload: true,
	panzoom: true,
	texts: {
		cue: "",
		hide: "",
		download: "",
		prev: "",
		next: "",
		link: "",
		zoom: "",
		zoomActive: "",
	},
	icons: {
		cue: "iconoir-add-circled-outline",
		hide: "iconoir-delete-circled-outline",
		download: "iconoir-download-circled-outline",
		prev: "iconoir-arrow-left-circled",
		next: "iconoir-arrow-right-circled",
		link: "iconoir-link",
		zoom: "iconoir-expand",
		zoomActive: "iconoir-collapse",
	},
	titles: {
		hide: "Close this image"
	}
});
iv.create();
